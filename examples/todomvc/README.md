todomvc-valuable
================

# TodoMVC - Valuable

This is a partial implementation of TodoMVC that uses Valuable for data and React for views.

Feel free to clone `valuable` and run it:

```bash
# clone valuable
cd valuable/examples/todomvc
npm install
npm run build
npm start &
open 'http://localhost:8080'
```

Then hit `Control-C` to kill.

## Development

Use `npm run build` to build the app.
